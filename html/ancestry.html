ANCESTRY REPORT

About: Using Python to load and analyze ancestry data from the Census Bureau and D3 to visualize the data - The purpose of this project was to view the dispersion of ancestry by congressional districts in the United States. The ancestries include West Balkan States: Albania, Croatia, Macedonia, Slovenia, Yugoslavia as well as Armenia, Turkish, and Greek country ancestries. This project began during my internship with Armenian lobbying NGO, ANCA in Washington DC in the summer of 2014.

Skills: Python, D3, HTML, CSS

Data: American Community Survey 2009-2013
